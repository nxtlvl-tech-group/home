How to download our apps

Step 1: Click "Download" on the app that you want. Step 2: If you have a PHP webserver installed on your device then you can simply save the file and run it on your localhost. If not though then open the file in notepad or a similiar device. Step 3: Copy the code from the software used and go to phpfiddle.org where you'll have to create an account. Paste the code and save the file with the same as the one you downloaded Step 4: Enjoy!
